About

About This Project

This project is Task 01 of the Prodigy InfoTech internship program. The goal is to build a responsive, interactive navigation menu using only HTML, CSS, and JavaScript.


The navbar listens to the scroll event and toggles a .scrolled class, switching the background from transparent to dark with a purple border.
